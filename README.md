# Readme
It is an exercise in using the VFI Toolkit, developed by Robert Kirkby. The code attempts to replicate the results of Chang et al. (MDY 2024). Please note that the code may contain bugs.
## Reference
Chang, Yongsung, Jong-Suk Han, and Sun-Bin Kim, Negative income tax and universal basic income in the eyes of Aiyagari, Macroeconomic Dynamics (2024), 28:813-825.
